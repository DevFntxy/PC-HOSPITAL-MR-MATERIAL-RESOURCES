# Directorio: Requisitos de Usuario (User Requirements - URs)

## 📁 Propósito de la Carpeta
Contiene la documentación que modela el sistema desde la perspectiva de las personas reales que operarán el Módulo MR en el hospital. Mapea las necesidades del personal administrativo, médico y de seguridad.

## 📄 Documentos que deben incluirse aquí:
- **`perfiles_usuario.md`**: Definición de *User Personas* (Administrador de Compras, Personal Médico, Guardia de Seguridad).
- **`matrices_necesidades.md`**: Listado de requerimientos declarados por los usuarios (ej. "El médico necesita localizar un ventilador en menos de 1 minuto").
- **`casos_uso_alto_nivel.md`**: Interacciones globales del usuario con el ecosistema de recursos materiales.
