# Directorio: Interfaces Gráficas de Usuario (GUIs)

## 📁 Propósito de la Carpeta
Raíz del diseño visual y de experiencia de usuario (UX/UI). Coordina cómo los datos estructurados del Módulo MR se transforman en pantallas legibles e interactivas para el personal del hospital.

## 📂 Organización del Directorio
Esta carpeta se divide estrictamente por canal de visualización del software:
- **`WebApp/`**: Espacio de diseño para la gestión administrativa pesada (escritorio/móvil).
- **`WerableApp/`**: Espacio de diseño para dispositivos de muñeca (Smartwatches para personal operativo).

## 📄 Documentos maestros aquí:
- **`sistema_diseno_branding.md`**: Paleta de colores institucionales, tipografías legibles y guías de contraste para entornos médicos.
- **`enlaces_prototipos.md`**: Accesos directos a los proyectos activos en Figma o Adobe XD.
