# Subdirectorio: Interfaz de la Aplicación Vestible (WerableApp UI)

## 📁 Propósito de la Carpeta
Contiene la documentación de diseño para las pantallas de relojes inteligentes del hospital. El foco de esta carpeta es el diseño bajo restricciones extremas de espacio físico.

## 📄 Documentos que deben incluirse aquí:
- **`interacciones_tactiles.md`**: Mapa de gestos soportados (swipes, double-taps para confirmación de alertas de stock crítico).
- **`UI_layouts_pantalla_reloj/`**: Diseños específicos para pantallas circulares o cuadradas (focalizados en la vista de alertas de seguridad y localización express de equipos médicos).
- **`especificaciones_ahorro_energia.md`**: Directrices de uso de negros puros (AMOLED) para mitigar el drenaje de batería durante las jornadas del personal médico.
