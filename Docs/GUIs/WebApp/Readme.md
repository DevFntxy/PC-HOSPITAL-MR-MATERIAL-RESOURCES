# Subdirectorio: Interfaz de la Aplicación Web (WebApp UI)

## 📁 Propósito de la Carpeta
Espacio exclusivo para almacenar los entregables de diseño de la plataforma Web del Módulo MR. Está orientada al trabajo del equipo Frontend enfocado en administración hospitalaria.

## 📄 Documentos que deben incluirse aquí:
- **`user_flows_administracion.md`**: Flujos de navegación para el alta de proveedores (Personas Morales) y consultas de auditoría financiera.
- **`wireframes_alta_fidelidad/`**: Carpeta con imágenes o layouts de las pantallas de dashboards analíticos y mapas de infraestructura física.
- **`inventario_componentes_web.md`**: Especificaciones de tablas de datos, modales de alerta y buscadores reutilizables.
