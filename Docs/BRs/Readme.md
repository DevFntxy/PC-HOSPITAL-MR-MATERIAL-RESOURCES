# Directorio: Requisitos de Negocio (Business Requirements - BRs)

## 📁 Propósito de la Carpeta
Este espacio está destinado exclusivamente a almacenar la documentación estratégica y legal que justifica la existencia del Módulo MR. Aquí se define el "por qué" de las reglas del hospital antes de escribir una sola línea de código.

## 📄 Documentos que deben incluirse aquí:
- **`objetivos_estrategicos.md`**: Alineación del módulo con la misión del hospital (evitar desabasto, mitigar pérdidas).
- **`reglas_negocio_hospitalarias.md`**: Políticas institucionales inmutables (ej. validación fiscal obligatoria de Personas Morales antes de compras).
- **`normativas_legales/`**: Carpetas con archivos sobre COFEPRIS o regulaciones de auditoría financiera aplicables al inventario médico.

## 🛠️ Lineamientos de Contenido
Cualquier cambio en las políticas de adquisición del hospital o restricciones de infraestructura debe impactar primero los archivos de esta carpeta.
