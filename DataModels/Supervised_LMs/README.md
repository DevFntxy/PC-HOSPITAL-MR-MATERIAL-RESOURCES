# Supervised_LMs

## Algoritmos matemáticos para la toma de decisiones

Los modelos supervisados utilizan datos etiquetados para aprender una función que hace predicciones o clasificaciones.

### Conceptos clave
- Datos de entrenamiento y prueba
- Variables de entrada (features) y salida (etiquetas)
- Ajuste de parámetros mediante minimización de error
- Validación cruzada para evaluar desempeño

### Algoritmos comunes
- Regresión lineal y polinómica
- Regresión logística
- Support Vector Machines (SVM)
- Redes neuronales supervisadas
- Árboles de decisión y bosques aleatorios

### Aplicaciones en toma de decisiones
- Predicción de riesgo y scoring
- Clasificación de resultados binarios o múltiples
- Recomendación basada en comportamiento histórico
- Optimización de recursos mediante pronósticos

### Matemáticas importantes
- Funciones de pérdida (MSE, cross-entropy)
- Gradiente descendente y variantes
- Regularización (L1, L2)
- Matrices y álgebra lineal para transformaciones
- Probabilidades y estadística para incertidumbre
