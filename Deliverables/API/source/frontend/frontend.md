# Frontend - Swagger UI

## Descripción

Esta carpeta representa la interfaz visual de documentación de la API mediante Swagger UI.

Swagger es una herramienta integrada en FastAPI que permite visualizar y probar los endpoints de manera interactiva desde el navegador, sin necesidad de utilizar aplicaciones externas como Postman.

La documentación se genera automáticamente a partir de las rutas y modelos definidos en el backend.

---

# Acceso a Swagger

Una vez ejecutada la API, Swagger puede visualizarse en:

```text
http://localhost:8000/docs
```
