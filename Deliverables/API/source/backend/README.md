# ⚙️ API Backend

Este directorio contiene la lógica backend de la API del sistema hospitalario.

## 📁 Contenido

* Endpoints y servicios REST.
* Controladores y lógica de negocio.
* Gestión de autenticación y seguridad.
* Modelos y acceso a base de datos.
* Configuración del servidor y dependencias.

## 🚀 Resumen Técnico

* Arquitectura orientada a servicios.
* Gestión centralizada de datos médicos.
* Integración con aplicaciones web y wearable.
* Estructura modular para facilitar mantenimiento y escalabilidad.
