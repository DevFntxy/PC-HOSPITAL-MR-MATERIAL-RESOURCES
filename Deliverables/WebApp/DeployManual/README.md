# 🌐 WebApp - Deploy Manual

Este directorio contiene la documentación técnica necesaria para el despliegue de la aplicación web del sistema hospitalario.

## 📁 Contenido

* Requisitos de instalación y entorno.
* Configuración del servidor y dependencias.
* Procedimientos de despliegue backend y frontend.
* Variables de entorno y configuración de servicios.
* Guías de mantenimiento y actualización.

## 🚀 Objetivo

Proporcionar instrucciones claras para la instalación, configuración y puesta en producción de la aplicación web.
