# 🌐 WebApp - Source

Este directorio contiene el código fuente completo de la aplicación web del sistema hospitalario.

## 📁 Estructura del Directorio

* **/backend:** Implementación de la lógica del servidor, API y gestión de datos.
* **/frontend:** Interfaz gráfica de usuario y componentes visuales de la aplicación.

## 🚀 Resumen Técnico

* Arquitectura separada entre frontend y backend.
* Comunicación mediante servicios REST.
* Código organizado para facilitar mantenimiento y escalabilidad.
