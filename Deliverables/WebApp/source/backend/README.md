# ⚙️ WebApp Backend

Este directorio contiene la lógica backend de la aplicación web del sistema hospitalario.

## 📁 Contenido

* API REST del sistema.
* Gestión de autenticación y seguridad.
* Conexión y administración de base de datos.
* Controladores, servicios y modelos.
* Configuración del servidor y dependencias.

## 🚀 Resumen Técnico

* Arquitectura orientada a servicios.
* Manejo centralizado de datos y procesos.
* Integración con servicios externos y módulos internos.
* Estructura modular para facilitar mantenimiento.
