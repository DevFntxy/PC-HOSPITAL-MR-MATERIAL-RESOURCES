# 🌐 WebApp - User Manual

Este directorio contiene el manual de usuario de la aplicación web del sistema hospitalario.

## 📁 Contenido

* Guías de navegación del sistema.
* Instrucciones de acceso y autenticación.
* Descripción de módulos y funcionalidades.
* Procedimientos de uso para usuarios administrativos y médicos.
* Recursos de apoyo y solución de problemas comunes.

## 🚀 Objetivo

Facilitar el uso correcto de la aplicación web mediante documentación clara y estructurada para usuarios finales.
